Examples¶

For Win10 Hibernate¶

; Wait for Hotkey
;   Ctrl + Win + Alt + l
; Send Hotkey
;   Ctrl + Win + x + u + s
^#<!l::Send #xuh

For Win10 Sleep¶

; Wait for Hotkey
;   Ctrl + Win + l
; Send Hotkey
;   Win + x + u + s
^#l::Send #xus

Home and End Hotkey¶

; Ctrl + Left
^Left::Send {Home}
; Ctrl + Right
^Right::Send {End}

Check for AHK Version and output message¶

If (A_AhkVersion < "1.0.39.00")
{
    MsgBox,20,,This script may not work properly with your version of AutoHotkey. Continue?
    IfMsgBox,No
    ExitApp
}
