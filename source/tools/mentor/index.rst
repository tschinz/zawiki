Introduction
============

Mentor Graphics is a company by Siemens and creator of two VHDL Design Tools:

* `Mentor Modelsim <https://www.mentor.com/products/fv/modelsim/>`_
* `Mentor HDL-Designer <https://www.mentor.com/products/fpga/hdl_design/hdl_designer_series/>`_

More information about the software can be found on the `SupportNet <https://support.sw.siemens.com>`_
