Introduction
============

JetBrains has different applications.

* `Intellij IDEA <https://www.jetbrains.com/idea/>`_ - General IDE can be used for any language, but mainly used for Java
* `PyCharm <https://www.jetbrains.com/pycharm/>`_ - Python specific
* ... I don't care about any other

It can be heavily extented with plugins found on the `Plugin Repository <https://plugins.jetbrains.com/>`_

See also :doc:`/coding/spinal/tools/intellij_idea` for Spinal specific use case
