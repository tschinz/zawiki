Libraries¶

Numeric Operations¶

Important

Never mix these Libraries!!!

Use numeric_std - it’s the proper IEEE standard and is more recent.

library ieee;
  use ieee.std_logic_1164.all;
  use ieee.numeric_std.all;

-- Never mix these libraries

library ieee ;
  use ieee.std_logic_1164.all ;
  use ieee.std_logic_arith.all ;
  use ieee.std_logic_unsigned.all ;
